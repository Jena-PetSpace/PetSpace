# 2026-03-31 작업 기록

## 개요
채팅 기능 고도화, 인증/계정 버그 수정, 온보딩 개선, UI 통일 작업을 진행했습니다.

---

## 1. 채팅 기능 고도화

### 채팅방 설정 개선
- 저장/취소 버튼 하단 배치 (가로 [저장] [취소])
- 저장 후 진입 경로에 따라 돌아가기 (채팅방 or 채팅 목록)
- 저장 내용 즉시 반영 (이름, 사진, 알림 설정)
- 알림/나가기 행 높이 통일
- 채팅방 사진: 카메라/갤러리 선택 + 저장 버튼 누를 때만 업로드

### 채팅 목록 UI
- 1:1 채팅방 인원수 숨김 (3명 이상만 표시)
- 알림 꺼진 채팅방 🔕 아이콘 표시
- 채팅방 이름 변경 실시간 반영 (목록 새로고침)
- displayName 우선순위: 커스텀 이름 > 상대방 이름 > 그룹 채팅
- displayAvatarUrl: 커스텀 사진 > 상대방 사진

### 채팅방 나가기
- 시스템 메시지: "OOO님이 채팅방을 나갔습니다" 전송
- RLS 에러 해결: 시스템 메시지 먼저 전송 → 나가기 처리
- 채팅방 이름 유지 (나가도 변경 안 함)
- 인원수 자동 갱신 (is_active=true 필터)

### 채팅 메시지 개선
- 카카오톡 스타일 묶어서 보내기 (image_urls TEXT[] 컬럼)
- 이미지 그리드: 2장(가로), 3장(1+2), 4장(2x2), 5+장(2x2+N)
- 읽음/안읽음: 1:1/그룹 모두 안 읽은 수로 통일
- 프로필 사진 상단 정렬
- 카메라/갤러리 선택 + 미리보기 + 여러 장 선택

## 3. 온보딩 개선

### 튜토리얼 스킵
- 반려동물 등록 → 감정분석 가이드 4페이지 건너뛰고 → 바로 완료 화면

### 완료 페이지 개선
- 스크롤 제거 → 한 화면에 맞춤 (Spacer)
- 아이콘 크기 축소 (200→140)
- 버튼 가로 배치: [홈으로 이동] [감정 분석]
- "언제든지 설정에서~" 텍스트 삭제
- "일상 기록" → "건강 및 일상 기록" 명칭 변경

### 감정분석 첫 진입 가이드
- 전체 화면 가이드 페이지 (1회만 표시)
- 브랜드 컬러 아이콘 + 4가지 촬영 팁 카드
- "분석 시작하기" 버튼으로 닫기
- SharedPreferences로 표시 여부 관리

## 4. UI 통일

### 사진 선택 공통 컴포넌트
- `ImageSourcePicker` 위젯 생성 (shared/widgets/)
- 앱 전체 11개 파일에 적용
- 명칭: "카메라로 촬영하기", "갤러리에서 선택하기"
- 아이콘: `camera_alt` (filled), `photo` (filled)
- 취소 버튼 제거 (바깥 터치로 닫기)
