## 4. 알림 시스템 변경

### 문제
FCM V1 API 서버 발송에 필요한 서비스 계정 키 생성이 조직 정책(`iam.disableServiceAccountKeyCreation`)으로 차단됨

### 해결
`PushNotificationService`를 Edge Function 호출 → **Supabase `notifications` 테이블 직접 INSERT** 방식으로 변경

```
좋아요/댓글/팔로우 발생
    ↓
PushNotificationService._send()
    ↓
notifications 테이블 INSERT
    ↓
Supabase Realtime 구독으로 인앱 알림 즉시 수신
```

- 자기 자신에게 알림 보내지 않는 가드 추가
- FCM 백그라운드 푸시는 서비스 계정 확보 시 Edge Function 재연결 예정
