## 4. 로고 작업

### 문제 파악
- 원본 `logo_petspace_img.svg`: 단색(#010101), 래스터 PNG 임베드 (발바닥), 2551×992px
- 로고 구조: `PetsPace` — `Space`의 `P` 카운터(속공간)에 발바닥 원형 내포

### 작업 내용
- `flutter_svg: ^2.0.10+1` pubspec 추가
- `assets/svg/` 폴더 신규 생성
- `logo_color.svg`: 딥블루 텍스트 + 코랄 발바닥 (라이트 배경용)
- `logo_white.svg`: 흰 텍스트 + 코랄 발바닥 (딥블루 배경용)
- `logo_mono.svg`: 딥블루 단색
- `PetSpaceLogo` 공유 위젯: `LogoVariant.light / dark / mono`
- iOS `LSApplicationQueriesSchemes` 추가 (kakaomap, comgooglemaps, nmap)
- Android `<queries>` 카카오맵 패키지 추가

### 잔여 이슈
- cairosvg가 원본 SVG의 다이아몬드 clipPath + 회전 matrix 렌더링 실패
- 원본 발바닥 PNG가 클립/회전 조합으로 인해 이미지 출력 시 깨짐
- **해결 방향**: 원본 SVG 구조를 유지하면서 색상만 직접 교체하는 방식으로 재작업 필요

## 5. 카카오맵 연동

- `hospital_search_page.dart` 전면 개선
  - 1순위: `kakaomap://` 딥링크 (앱 설치 시 + 위치 포함)
  - 2순위: `map.kakao.com` 웹 URL (앱 미설치 시)
  - 3순위: `google.com/maps` 최종 fallback
- 위치 권한: `geolocator: ^13.0.2` 추가

## 6. 정현 담당 인프라 작업 (미완료)

아래 항목은 코드 작업이 아닌 Supabase 콘솔/스토어 직접 설정 필요:

| 항목 | 내용 |
|------|------|
| **Supabase 테이블** | `user_points`, `user_quests`, `user_badges`, `pet_follows` 생성 필요 |
| **DB 인덱스** | `posts(author_id, created_at)`, `notifications(user_id, is_read, created_at)` SQL 실행 |
| **Storage RLS** | `profiles/%userId%/*`, `posts/%userId%/*` owner DELETE 정책 |
| **소프트 삭제** | `posts`, `comments` 테이블 `deleted_at TIMESTAMPTZ` 컬럼 추가 |
| **FCM Edge Function** | 주간 리포트·건강 알림 실발송용 함수 배포 |
| **Google Maps API** | 현재 url_launcher 방식으로 동작, SDK 방식 전환 원하면 키 발급 필요 |
| **Apple/Google 인앱결제** | 프리미엄 구독 연동 시 스토어 설정 필요 |
