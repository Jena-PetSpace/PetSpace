## P0-1: 홈 화면 카테고리 및 데이터 연결

### 1a. 카테고리 순서 변경
- **변경 전**: 인기 → 매거진 → 커뮤니티 → 건강 → 훈련
- **변경 후**: 인기 → 커뮤니티 → 건강 → 훈련 → 매거진
- **파일**: `category_filter_chips.dart`, `home_page.dart`

### 1b. 피드 커뮤니티 탭 카테고리 수정
- **변경 전**: 전체 / 건강Q&A / 사료추천 / 자유Q&A / 정보공유 / 자유 (FilterChip + 체크표시)
- **변경 후**: 전체 / Q&A / 건강 / 훈련 / 매거진 (단순 버튼 활성화)
- `community_posts` 테이블(미존재) 대신 `posts` 테이블의 `hashtags` 기반 필터링으로 변경
- **파일**: `feed_hub_page.dart`

### 1c. 홈 더보기 → 피드 커뮤니티 해당 탭 이동
| 홈 카테고리 | 더보기 이동 경로 |
|------------|----------------|
| 커뮤니티 | `/feed?tab=community` |
| 건강 | `/feed?tab=community&category=health` |
| 훈련 | `/feed?tab=community&category=training` |
| 매거진 | `/feed?tab=community&category=magazine` |
| 인기 배너 | `/feed?tab=community` |

- `app_router.dart`에 `initialCategory` 파라미터 추가
- **파일**: `community_preview.dart`, `magazine_grid.dart`, `hot_topic_banner.dart`, `app_router.dart`

### 1d. 더미 데이터 → 실데이터 교체
- `community_preview.dart`: 하드코딩 제거 → `posts` 테이블에서 실시간 로딩
- `magazine_grid.dart`: 하드코딩 제거 → `posts` 테이블에서 `magazine` 태그 기반 로딩
- 게시글 클릭 시 `/post/{postId}`로 이동 (기존: 클릭 불가 또는 커뮤니티 탭으로만 이동)
- `hot_topic_banner.dart`: 클릭 가능하게 수정 (기존: 클릭 불가)
- `home_page.dart`: 건강/훈련 더미 콘텐츠 메서드 삭제 → `CommunityPreview(category:)` 위젯으로 대체

### 관리자 초기 게시글 (Seed Data)

  - 건강 2개: 겨울건강, 예방접종

## P0-2: 프로필 팔로워/팔로잉 목록 연결

### 문제
- 프로필 페이지의 팔로워/팔로잉 탭이 "곧 추가됩니다!" 스텁만 표시

### 수정
- `follows` 테이블에서 실시간 데이터 로딩 (FutureBuilder)
- 팔로워: `follows.following_id = userId` → `follower` 유저 정보 조인
- 팔로잉: `follows.follower_id = userId` → `following` 유저 정보 조인
- 프로필 이미지, 이름 표시 + 탭 시 해당 유저 프로필로 이동
- 빈 목록 시 "아직 팔로워/팔로잉이 없습니다" 표시
- 메시지 버튼: 스텁 → 채팅 페이지 이동
- **파일**: `profile_page.dart`
